## Installation
Add this line to your application's Gemfile:

```ruby
gem 'sakura_rails'
```

And then execute:
```bash
$ bundle install
```

Or install it yourself as:
```bash
$ gem install sakura_rails
```

app/assets/stylesheets/application.css

```bash
@import 'sakura_rails';
 ```

view_file
```bash
<%= fall_sakura %>
```
